# Next.js demo

This is a demo of a Next.js application. It showcases the following
Next.js\Vercel features:

- Server Actions
- Dynamic Routes
- Search and Pagination
- Error Handling
- Streaming
- Authentication
- Authorization
- Deployment
- NeonDB integration (through Vercel)

Deployment: <https://next-js-demo-2-two.vercel.app/>
